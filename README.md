# Alpha-s-Gym-Landing-Page

## Definición de Problema:

 

El problema a resolver se asocia con el gimnasio de un amigo de mi ciudad natal (Ponce Enríquez).

El gimnasio depende de métodos tradicionales (publicidad de boca en boca) para atraer clientes. Los interesados tienen que ir a preguntar personalmente, lo que resulta en un embudo de ventas ineficiente y una alta fricción para obtener información clave (precios, horarios, ofertas).

La necesidad de mi amigo sería el de capturar leads (clientes potenciales) calificados y promocionar una oferta de prueba irresistible.

Con todo esto el objetivo de mi landing page sería el de conseguir que el usuario agende un par de días de prueba gratuita y luego mostrar los planes de suscripción.
